# Sistema de Gestão de Doações para ONG 

 

## 📌 Descrição 

Este projeto tem como objetivo desenvolver um sistema para gerenciamento de doações de uma ONG, permitindo o cadastro de doadores, registro de doações, controle de itens e distribuição para beneficiários. 

 

## 🎯 Objetivo 

Facilitar o controle e organização das doações, garantindo maior transparência e eficiência no processo. 

 

## 👥 Público-Alvo 

- Voluntários 

- Coordenadores da ONG 

 

## ⚙️ Funcionalidades 

- Cadastro de usuários 

- Cadastro de doadores 

- Registro de doações 

- Controle de itens doados 

- Gestão de beneficiários 

- Relatórios e dashboard 

 

## 🚀 Tecnologias (Stack) 

- Frontend: HTML, CSS, JavaScript 

- Backend: Node.js (ou Java / PHP) 

- Banco de Dados: MySQL
